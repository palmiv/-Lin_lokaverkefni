### Velkominn í Lokaverkefni í linux.

Í þessu lokaverkefni er ég búinn að gera tutorial. Í þessum tutorial kemur framm hvernig maður setur upp ubuntu server, hvernig þú setur upp docker container, mysql, PHPMyAdmin.

Ubuntu server

Hægt er að sækja image-ið hérna: http://www.ubuntu.com/download/server . og skalt þú þá ná í það og keyra Installation. Veldu SSH Client og hallt þú svo bara áfram

Apache

Þegar þú hefur lokið install'ið á Ubuntu ertu kominn í næsta skref setja upp Apache. Byrjum á því að keyra sudo apt-get update sudo apt-get upgrade sudo apt-get install apache2

Til að sjá IP töluna ykkar skalltu keyra ifconfig eth0 | grep inet | awk '{ print $2 }'

Smá um apache: Default port sem Apache hlustar á er 80 og á https er það 443.

Til að kveikja á Apache vefþjónin notar þú sudo service apache start Og að slökkva sudo service apache stop

MySQL

MySQL er eitt mest notaði gagnagrunnsforrit í dag. Til að setja upp MySql skallt þú keyra inn þessa skipun sudo apt-get install mysql-server þá er beðið þið um að gera root password. Gott er að vera með bæði öruggt og eitthvað auðvelt að muna. Ef þú hefur enga reynslu með MySql mæli ég með að læra basic hér

Docker

Nú þegar við höfum install'að mysql þá förum við í næsta og það er Docker. Til að install'a Docker þá keyrum við inn eftifarandi kóða:

docker pull httpd docker pull eboraas/apache docker run -p 80:80 -p 443:443 -v /home/user/mysite/:/var/www/html/ -d eboraas/apache

Nú þegar við erum komin með docker ætlum við að gera scriptu sem býr til sql notendur: touch dockerscripta sudo chmod 777 dockerskripta nano dockerscripta

það sem þetta gerir er það að þú býrð til skriptuna. Gefur henni 777 leyfi sem er öll leyfin sem þú getur gefið. Síðast er það Nano sem gerir þér kleyft að edita skriptuna. Næst skaltu skrifa inní skriptuna ´for i in {1..15} do mysql -uroot -proot -e "CREATE USER 'notandi_$i'@'localhost' IDENTIFIED BY 'password_$i'" done´

Næst gerum við töflu sem heitir Logs. Við ætlum að nota basic mysql skipun í það.

mysql -u root -p //svo skrifaru inn passwordið þitt

CREATE DATABASE lokaverkefni;

USE lokaverkefni;

CREATE TABLE IF NOT EXISTS Logs(

 ID int(5) NOT NULL AUTO_INCREMENT,

 Date Date DEFAULT NULL,

 User varchar(20) DEFAULT NULL,

 Memory varchar(20) DEFAULT NULL,

 IP varchar(20) DEFAULT NULL,

 Hostname varchar(20) DEFAULT NULL,

 PRIMARY KEY(ID)

);
